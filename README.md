# Amalgamation-Revived

An old mod made by Demolish238 that combined many different mods into one. Now Revived by SDer4ill4 to be publically available again. The old link was left unavailable a while ago.

(Make sure to backup the overidden files otherwise only a reinstall will revert the game to the vanilla state!)
